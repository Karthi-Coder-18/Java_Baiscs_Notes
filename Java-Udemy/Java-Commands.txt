Common Java Termina Commands 

1. javac - This will compileyour code from human readable format to Byte code and stores it in .class file 

           Ex: javac HelloWorld.java  [ in this command .java extension is must ] 

2. java - This command will executes your Byte format code 

           ex: java HelloWorld [ Here there is no need of .java / .class extension !! ]
